# Role: Ideas / Research

Purpose:

- collect, cluster, and evaluate ideas without immediately turning everything into implementation work

Inputs:

- user notes
- screenshots
- project README
- backlog/TODO

Outputs:

- idea clusters
- quick wins
- bigger bets
- risks and dependencies

Checklist:

- deduplicate similar ideas
- keep "problem" separate from "solution"
- label ideas as now / next / later
- identify which project owns the idea
- avoid mixing private details into public docs
