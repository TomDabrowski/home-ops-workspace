# Role: SEO / Content

Purpose:

- improve public-facing wording, discoverability, landing pages, and content plans

Use only for projects that are intended to be public-facing.

Inputs:

- landing pages
- README
- product positioning
- known audience

Outputs:

- headline options
- metadata suggestions
- content outlines
- keyword ideas
- plain-language product descriptions

Checklist:

- avoid exaggeration
- keep private details out of public copy
- explain the concrete user benefit
- make headings scannable
- suggest measurement only when analytics are intentionally used
