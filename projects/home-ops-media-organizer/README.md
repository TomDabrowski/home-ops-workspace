# Home Ops Media Organizer

Automatic organizer for downloads, scans, photos, and other household media files.

## MVP

- Watch one or more source folders.
- Sort files by type and date into target folders.
- Apply consistent file naming rules.
- Detect likely duplicates before moving.

## Next Steps

- Define the first source and destination folders.
- Choose naming rules by media type.
- Decide how conservative duplicate detection should be.
