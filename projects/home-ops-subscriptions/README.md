# Home Ops Subscriptions

Subscription and invoice tracker for recurring costs, deadlines, and reminders.

## MVP

- Store subscriptions, invoices, due dates, and cancellation periods.
- Show monthly and yearly recurring cost totals.
- Highlight upcoming renewals and overdue items.
- Export a simple overview for later finance integration.

## Next Steps

- Define the core entities for subscriptions, invoices, and reminders.
- List the manual fields needed for the first working version.
- Plan the later handoff into the finance tracker.
