# Home Ops Backup Control

Backup verification tool for NAS and local backup jobs.

## MVP

- Check whether backup targets are reachable.
- Track last successful backup timestamps.
- Flag stale or failed backup jobs.
- Produce a small daily status summary.

## Next Steps
