# Home Ops Dashboard

Unified household dashboard that can surface finance, home lab, and household status.

## MVP

- Show a small daily dashboard with finance, tasks, and service status.
- Display current alerts from other tools.
- Work well on desktop and tablet layouts.
- Stay useful even before all modules exist.
